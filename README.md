# Diabetes Healthcare Data Analysis using Python and SQL

This project performs healthcare data analysis on diabetes dataset using Python, Pandas, SQLite and Matplotlib.

## Project Steps
- Load dataset using pandas
- Explore dataset information
- Store data in SQLite database
- Perform SQL queries for analysis
- Create visualizations using matplotlib
- Implement OOP for reusable analysis

## Technologies Used
- Python
- Pandas
- SQLite
- Matplotlib
- Google Colab

## Dataset
Pima Indians Diabetes Dataset

## Project Overview

This project analyzes a healthcare diabetes dataset to understand patterns related to glucose level, BMI, age and diabetes outcome.

The dataset was stored in SQLite database and multiple SQL queries were used to extract meaningful insights.

Data visualization was performed using Matplotlib to identify relationships between medical attributes and diabetes condition.

This project demonstrates how Python and SQL can be used together for basic healthcare data analytics.
